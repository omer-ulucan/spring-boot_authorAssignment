# Spring-Boot Author
## Target of the Project</br>
The management of authors in this Java Spring project is handled through the use of a RESTful API.</br></br>

## What Dependencies We Used in This Project?
✔️Spring Web</br>
✔️Spring JPA</br>
✔️Spring DevTools</br>
✔️H2</br>
✔️Lombok</br>
</br></br>
